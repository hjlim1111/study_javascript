; (Quantifiers)

* + ?

aabc abc bc *(별표) : 별표 앞에 등장하는 패턴이 몇번 등장하는 가에서 0~여러개 // 있을 수도 있고 없을 수도 있음 regular expression : a*b first match : aabc abc bc all match : aabc abc bc +(더하기) : 1~여러개 //하나도 없을 수 없음. 1이상이여야 함 regular expression : a+b first match : aabc abc bc all match : aabc abc bc ?(물음표) : 없거나~1개 regular expression : a?b first match : aabc abc bc all match : aabc abc bc
